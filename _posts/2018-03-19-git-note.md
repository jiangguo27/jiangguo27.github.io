### SSH Key
> 
	$ ssh-keygen -t rsa -b 4096 -N ''

生成公钥，添加到Git Server上，避免每次和Server通信都要输入密码。

### Alias
> 
	[alias]
		st = status
		co = checkout
		ci = checkin
		br = branch

添加以上内容到~/.gitconfig，可以用缩略名来替换。

### Proxy
>	
	$ git config --global http.proxy 'proxy_ip:port'
	$ git config --global https.proxy 'proxy_ip:port'
